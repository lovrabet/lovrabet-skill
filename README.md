# Lovrabet Skill

面向 [skills](https://www.npmjs.com/package/skills) 开放生态的 **Lovrabet（云兔）** Agent Skill 发布仓库。

## 安装

```bash
npx skills add lovrabet/lovrabet-skill
```

## 本仓库结构

* 根目录：`LICENSE`（MIT）、本说明（`README.md`）
* Skill 本体：`skills/lovrabet/`（`SKILL.md`、guides、元数据等）

这样根目录保留仓库说明与许可证，与 `skills/` 下的技能包内容分开，便于浏览与区分。
