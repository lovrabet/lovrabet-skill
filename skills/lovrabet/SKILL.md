# Lovrabet Skills

本 SKILL.md 已过时，最新 Skills 已发布至 GitHub。

## 安装最新 Skills

请执行以下命令安装：

```bash
npx skills add lovrabet/rabetbase -g -y
```
需要跟`rabetbase-cli`合并使用，才能获得完整的开发能力
安装命令:

```bash
npm install -g @lovrabet/rabetbase-cli@latest
```

仓库地址：https://github.com/lovrabet/rabetbase

安装后将自动获得最新的 Lovrabet 开发工作流 Skills。
